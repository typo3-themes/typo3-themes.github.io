Finally it's time to present our work.

We are currently working on an extension which allows to create, install and manage themes in TYPO3.

To achieve that we talked to a lot of people to find a standard for a strong packaging standard.

Together with some other extensions you can easily create flexible configurable themes, which can be transfered easily between TYPO3 Installations.

TYPO3 Conference 2013 Germany

Keep it simple

Each part of the theme framework can be used standalone, there is no need to add heavy dependencies to your themes.

We base on strong proven technologies which are in the TYPO3 core since a long time.

We try searched and combined the best and strongest technologies available in the TYPO3 world and combined them to get a quite new experience.
